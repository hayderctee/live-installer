package managers and more !!

Uruk GNU/Linux come with Package Managers Simulator, This program can simulate a famous package managers Commands this mean you can install, uninstall, update, remove the Packages useing any Package Manager you want like "pacman", "yum', "zypper" .... etc.
and with Uruk GNU/Linux Building software from source by one command : u-src PACKAGE_SOURCE
and install RPM packages on your system also with one command : u-rpmi "PACKAGE_NAME"
